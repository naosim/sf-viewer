フィルターの入力方法

対応演算子

==等しい
!=等しくない
> >= < <=数値比較
AND / &&論理積
OR / ||論理和
INリストに含まれる
NOT INリストに含まれない
LIKE部分一致（%がワイルドカード）
IS NULL空値判定
IS NOT NULL非空値判定
DateTime('YYYY-MM-DD')日付
DateTime('YYYY-MM-DDTHH:MM:SS')日付時刻

使用例

ProductCode IN ('GC1060', 'GC1020')
ProductCode == 'GC1060' AND IsActive == true
Name LIKE '%Diesel%'
Description IS NULL
(Status == 'Active' OR Status == 'Pending') AND IsActive == true
CreatedDate > DateTime('2024-01-01')
LastModifiedDate >= DateTime('2025-01-01T09:00:00')
※ 日付の ==, != は未対応です。範囲指定を使用してください。
例: CreatedDate >= Date('2026-04-22') AND CreatedDate < Date('2026-04-23')
